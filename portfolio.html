Sistema de Gestão de Pizzaria

API REST desenvolvida para gerenciamento de pedidos, usuários e operações de uma pizzaria. O projeto foi criado com foco em boas práticas de desenvolvimento backend, autenticação de usuários e gerenciamento do banco de dados através de migrações.

Tecnologias utilizadas:

Python FastAPI SQLAlchemy MySQL JWT Alembic Passlib Argon2

Principais funcionalidades:

• Cadastro e autenticação de usuários
• Proteção de rotas com JWT
• Gerenciamento de pedidos
• Controle de dados persistidos em banco de dados
• Migrações de banco com Alembic

Principais conceitos aplicados:

• Autenticação e autorização
• Hash seguro de senhas com Argon2
• Gerenciamento de migrações
• Modelagem de banco de dados relacional
• Arquitetura de APIs REST
• Organização de projeto em camadas

Sistema de Gerenciamento de Tarefas

API REST desenvolvida com FastAPI para gerenciamento de tarefas com autenticação de usuários. O sistema permite cadastro, login e gerenciamento de tarefas individuais, garantindo que cada usuário tenha acesso apenas aos seus próprios dados.

Tecnologias:

Python FastAPI SQLAlchemy JWT MySQL

Principais funcionalidades:

• Cadastro de usuários
• Login com autenticação JWT
• Criação, edição e exclusão de tarefas
• Proteção de rotas
• Associação entre usuários e tarefas

Principais conceitos aplicados:

• Autenticação e autorização
• Tokens JWT
• Relacionamentos ORM
•Estruturação de APIs REST
